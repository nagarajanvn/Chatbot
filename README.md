# AI Chat App

A modern ChatGPT-like chat application built with React, TypeScript, Vite, Tailwind, and Groq.

## Features

- Responsive chat UI
- Markdown rendering
- Syntax highlighted code blocks
- Local conversation history
- Theme toggle
- Groq API integration

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy the environment file:
   ```bash
   cp .env.example .env
   ```
3. Add your Groq API key to `.env`.
4. Start the app:
   ```bash
   npm run dev
   ```

## Structure

- client: Vite React frontend
- server: Express backend
